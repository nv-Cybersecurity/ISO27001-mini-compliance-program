# ISO 27001 Mini Compliance Program – SecureHealth SaaS

## Overview
This project simulates building a compliance program for a healthcare SaaS company handling sensitive patient data (PHI).

## Framework
ISO 27001

## Scope
- AWS Cloud Infrastructure
- Web Application & APIs
- Patient Data Storage
- Employee Endpoints

## Deliverables
- Control Matrix (15+ controls)
- Risk Register
- Compliance Overview Document
- Access Control Policy
- Internal Audit Report

## Key Skills
- GRC (Governance, Risk, Compliance)
- ISO 27001 Controls
- Risk Assessment
- Gap Analysis
- Policy Development

## Summary
This project identifies security gaps and provides remediation strategies to improve compliance readiness and security posture.
